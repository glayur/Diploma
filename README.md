
## Установка и воспроизведение анализа

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/glayur/Diploma.git
   cd Diploma
   ```

2. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   source venv/bin/activate    # Linux/Mac
   venv\Scripts\activate       # Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запускайте ноутбуки **в следующем порядке**:
   - `drop_empty_osm.ipynb` (при необходимости)
   - `blocksnet.ipynb`
   - `kriterii.ipynb`
   - `index.ipynb` (главный расчёт)

**Примечание:** В ноутбуках могут встречаться абсолютные пути к файлам. При необходимости замените их на относительные.
